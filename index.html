~ Tasks for Coders ~

You'll need to complete all the tasks
Coders
  • Roomintro
    - Any title image
    - Any background image
    - Room name is Guardians of Life
    - Green buttons (any shape), black border
    - 3 buttons: Room Auth, Discord and Room Events
    - A marquee
  • Declare
    - Based off your very own trainer card. Must contain Background images, buttons and a title image.
  • Trainer Card
    - A room event is being held in a room named Fairy Land! Make a global declare for it. Must contain a button linking to the room, background image, information and room name as title. You can use /testdeclare [ your code ] to test your declares.

After completing all the tasks, you'll need to contact our coders either on Discord or Dawn.
